# RawKit

macOS RAW 图像预览和处理工具,支持 Sony A7R4、Sigma DPQ 等相机的 RAW 格式以及常规图像格式。

## 功能特性

### 图像格式支持
- RAW 格式: ARW (Sony)、X3F (Sigma)、CR2/CR3 (Canon)、NEF (Nikon)、DNG 等
- 常规格式: JPEG、PNG、TIFF

### 图像查看
- 缩放和平移操作(支持手势)
- 双击重置缩放
- 快速缩略图生成
- 文件元数据显示

### 实时图像调整
- **基础调整**
  - 曝光调整
  - 亮度控制
  - 对比度调整
  - 高光恢复
  - 阴影提升
- **色彩调整**
  - 饱和度
  - 自然饱和度
  - 色温调节
  - 色调微调
- **细节调整**
  - 锐化

### 文件管理
- 拖放文件支持
- 批量导入
- 文件夹浏览

## 技术栈

- Swift 6.0
- SwiftUI (macOS 14+)
- Core Image (实时图像处理)
- CGImageSource (RAW 解码)
- Actor 并发模型

## 构建和运行

1. 打开 Xcode 项目:
   ```bash
   open RawKit.xcodeproj
   ```

2. 选择目标设备为 "My Mac"

3. 点击运行按钮 (⌘+R) 或 Product > Run

## 系统要求

- macOS 14.0+
- Xcode 15.0+ (用于开发和编译)
- 8GB+ RAM (推荐用于处理大型 RAW 文件)

## 使用说明

1. 启动应用后,会自动扫描图片文件夹
2. 点击"添加文件"或"打开文件夹"导入图像
3. 从左侧列表选择图像进行查看
4. 使用右侧调整面板进行实时图像调整
5. 使用鼠标滚轮或双指手势缩放图像
6. 拖动图像进行平移
